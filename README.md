# Anuj-Tech-Support-V2

This project is a Java-based application that calculates the total price of sushi plates based on plate colors. 

It demonstrates Object-Oriented Programming (OOP) principles and multiple design patterns, including Factory, Strategy, and Observer.

The system also supports discount calculation and bill splitting among multiple users, simulating a real-world dining scenario.

## ✨ Features
- Multiple plate types with different prices
- Automatic total price calculation
- Discount pricing strategy
- Bill splitting among multiple users
- Dynamic object creation using Factory Pattern
- Flexible pricing using Strategy Pattern
- Real-time updates using Observer Pattern

- ## 🧠 Design Patterns Used
- **Factory Pattern** – Used to create plate objects dynamically
- **Strategy Pattern** – Used to handle different pricing methods
- **Observer Pattern** – Used to update bill changes in real-time

- ## 🧩 OOP Concepts Applied
- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

## 🚀 How to Run

1. Compile the project:

   javac *.java

2. Run the server:
   
   java WebServer

3. Open browser:

   http://localhost:8000
